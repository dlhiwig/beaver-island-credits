## Filing Requirements

### Who Files
The cooperative files Form 1099-B as the barter exchange. Individual members do NOT file 1099-B — they use the info to report barter income on their personal Form 1040 (Schedule 1).

### When
- **January 31:** File with IRS + furnish copies to members
- **Extension:** Form 8809 for automatic 30-day extension (filing only, not furnishing)
- **10+ forms:** Must file electronically via IRS FIRE system

### Threshold
- Report if member's total FMV ≥ $600/year (aggregate all transactions)
- Under $600: No 1099-B required, but member still self-reports income on taxes

## How to Fill Out Form 1099-B

| Box | Content |
|-----|---------|
| **1a** | Description of property ("2 cords firewood traded for services") |
| **1b** | Date of sale/exchange |
| **1d** | Total FMV of all barter exchanges for this member (USD) |
| **2** | Cost basis ($0 for most barter) |
| **3** | ☑ Check "barter exchange" |
| **4** | Federal tax withheld ($0 typically) |
| **Payer** | Cooperative name, address, EIN |
| **Recipient** | Member name, address, TIN/SSN |

## Determining Fair Market Value (FMV)

| Good/Service | Method | Example |
|-------------|--------|---------|
| Firewood | Local market price per cord | $200/cord → 2 cords = $400 |
| Fish | Market rate per pound | $10/lb salmon |
| Home repairs | Local hourly rate | $100/hr mechanic |
| Medical supplies | Retail price | $50 first-aid kit |

**Rules:**
- Use comparable sales (Craigslist, local ads, industry rates)
- Document sources for audit defense
- Use values at time of exchange (not later)
- Don't undervalue to avoid reporting — IRS will challenge

## Sample Scenarios

### Vet A: 2 cords firewood ($400) → 400 credits
- FMV: $400
- If only trade this year → under $600 → no 1099-B
- If Vet A has other trades totaling $200+ → report ALL trades

### Vet B: Spends 100 credits on boat engine repair ($100)
- FMV: $100
- Add to Vet B's annual total
- Report if aggregate > $600

### Vet C: Provides medical supplies for 50 credits ($50)
- FMV: $50
- Include in annual total

## Penalties for Non-Compliance

| Violation | Penalty |
|-----------|---------|
| Filed within 30 days late | $50/form (max $588,500) |
| Filed 30 days – Aug 1 late | $130/form (max $1,886,500) |
| Filed after Aug 1 | $310/form (max $3,783,000) |
| Intentional disregard | $270/form (no cap) |
| Wrong TIN | $50/form + backup withholding |

*Small business cap (gross receipts <$5M): $588,500/year*

---

## EIN Setup

1. Go to IRS.gov → "Apply for an EIN"
2. File Form SS-4 online (free, takes minutes)
3. Use EIN on all 1099-B forms and IRS correspondence

## Barter Exchange vs. Informal Barter

| Feature | Organized Exchange (IRC 6045) | Informal 1-on-1 |
|---------|-------------------------------|-----------------|
| Structure | Cooperative facilitates trades | Direct person-to-person |
| Reporting | Cooperative files 1099-B | No 1099-B required |
| Taxes | Members report income | Members report income |
